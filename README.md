# random-cat-noise
Outpus a random cat noise.
## Installation
`npm i random-cat-noise`
## Supported noises
```
const noises = [
    'Meow',
    'Purr',
    'Chirp',
    'Hiss',
    'Yowl',
    'Growl',
    'Howl',
    'Chatter',
    'Trill',
    'Chirrup',
    'Mew',
    'Pip',
    'Yelp',
    'Screech',
    'Squeak',
    'Squawk',
    'Hiss',
    'Roar',
    'Scream',
    'Caterwaul',
    'Bellow',
    'Whine',
    'Whimper',
    'Yap',
]
```
